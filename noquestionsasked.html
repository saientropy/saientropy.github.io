This is a public commitment for me to learn computer science in a disciplined and non-result driven way.

I first started coding in 2016 (age 16), the first time I used a computer was 2006 (age 6). For 10 years I had absolutely zero curiosity to learn how computers worked until I was forced to learn coding in school. I hated every single minute of Turbo C++. Then I was forced to learn C and Python during college ( 2017) and I did not like that one bit. I would rather bite my finger nails off than write code.
I somewhat enjoyed "coding" when I was working on project that made a robot play soccer using vision but coding was my least favorite part in that process. I would rather open up servo motors to see how it works and what makes it work.

I like hardware, I like opening things up and putting them back. But I'm way too young to say no to learning almost anything. This is a public commitment to learn computer science from a physics first perspective.

I will take this 2 ways.

  1. Learn all the way from silicon to making my own CPU to god knows where that will lead to
  2. start with a programming language. For me its python and C for now since I have some familiarity.
    I don't think I will ever be a software engineer or any kind of engineer related to computer science but I want the power to make my life easier through making machines do work for me. No questions asked.
